# DevOps Docker CI/CD Project

## Description
This project demonstrates basic DevOps practices, including containerization and CI/CD automation.

A simple Python-based web server is containerized using Docker and automatically built and executed using GitHub Actions on each code push.

## Technologies
- Docker
- GitHub Actions (CI/CD)
- Python

## Features
- Containerized web server
- Port mapping and basic networking
- Automated build and execution pipeline
- Consistent runtime environment

## How to Run Locally

Build the Docker image:
```bash
docker build -t devops-app .
